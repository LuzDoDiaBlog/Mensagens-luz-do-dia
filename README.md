# Setup Instructions for VITE_GEMINI_API_KEY

In order to run the application, you need to set up the `VITE_GEMINI_API_KEY` environment variable. Follow these steps:

## Setting Up Environment Variable

1. Create a file named `.env` in the root of your project directory if it doesn't exist.
2. Open the `.env` file in your text editor.
3. Add the following line:
   
   ```
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. Replace `your_gemini_api_key_here` with your actual Gemini API key.
5. Save the file.

## Note
Ensure that the `.env` file is listed in your `.gitignore` file to avoid committing sensitive information to the repository.
